Quadratic function
A quadratic function is a function of the form f(x) = ax^2 + bx + c or y=ax^2 + bx + c, where a, b and c are real numbers and a ≠ 0.
Examples of Quadratic functions
A quadratic function has a variable(mostly x or y) to the second power. An example would be x^2 - 4x + 2 or 2x^2 + 8x + 4. As long as it is the form ax^2 + bx + c it is a quadratic function.
Graph of a Quadratic function
The graph of a quadratic function is a parabola. If the parabola opens upward, the lowest point called the minimum point is the vertex of the parabola. If the graph opens upward, the vertex is at the highest point called the maximum point.
Graphing Quadratic functions
A parabola will open upward and have a minimum point when the coefficients of y and x^2 have the same sign. It will open downward and have a a maximum point when the coefficients of y and x^2 have opposite signs.
The range of a Quadratic function
The range refers to all possible values of y. To find the range, determine the vertex. If the parabola opens up, the range includes the y-value of the vertex and all numbers greater than it. If the parabola opens down, the range includes the y-value of the vertex and all numbers less than it.
3 Quadratic forms
The Standard form is one, written as f(x) = ax^2 + bx + c. Factored form is another, which is written as f(x) = (x - p)(x - q). A third form is Vertex form, written as f(x) = a(x - h)^2 + k.
Linear function
A linear function is a function of the form f(x) = mx + b or y = mx + b where m and b are real numbers. Its is a line with slope m and y-intercept b.
Linear function examples
A linear function is a function that represents a straight line on the coordinate plane. For example, y = 3x - 2 represents a straight line on a coordinate plane and hence it represents a linear function. Since y can be replaced with f(x), this function can be written as f(x) = 3x - 2.
Range of a Linear function
Linear functions (almost) always have infinite domains and ranges. The exception is when the graph is a horizontal line. This happens for functions that equal a constant such as f(x) = b. These functions have infinite domains but a range that has only one value, b.
Forms of Linear functions
There are also three(3) forms of of the linear equation, these are the slope-intercept form that has a form y = mx + b, where m is the slope and b is the y-intercept, the standard form with the form Ax + By = C, where A, B, and C are integers and A ≥ 0. A and B can be zero but not both at the same time, and finally the general form with the form Ax + By + C = 0,, where A or B can be zero but not both at the same time.
Rational function
A rational function is a function that is the ratio of polynomials. Any function of one variable, x, is called a rational function if, it can be represented as f(x) = p(x)/q(x), where p(x) and q(x) are polynomials such that q(x) ≠ 0.
Exmaples of Rational function
As stated above, rational functions take the form p(x)/q(x) and q(x) cannot be equal to 0. For example, f(x) = (x^2 + x - 2)/(2x^2. - 2x -3) is a rational function since the value q(x) is not 0.
